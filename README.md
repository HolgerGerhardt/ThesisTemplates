# Thesis Templates


## General Information

This repository includes a LaTeX template and a Microsoft Word template for bachelor’s/master’s theses and term papers.

The formatting used in the templates corresponds to the requirements specified by the examination office of the Department of Economics at the University of Bonn. (See <a href="https://www.econ.uni-bonn.de/examinations/de/informationen/bachelor/bachelorarbeit" rel="noopener noreferrer" target="_blank">https://www.econ.uni-bonn.de/examinations/de/informationen/bachelor/bachelorarbeit</a> and <a href="https://www.econ.uni-bonn.de/examinations/en/information/master-economics/master-thesis" rel="noopener noreferrer" target="_blank">https://www.econ.uni-bonn.de/examinations/en/information/master-economics/master-thesis</a>, respectively.)

## Noteworthy Features

Word does not store custom caption labels in the document itself but only in the local Normal.dotm template (see <a href="https://wordribbon.tips.net/T012870_Passing_Custom_Caption_Labels_from_a_Template_to_a_Document_Based_on_the_Template.html" rel="noopener noreferrer" target="_blank">https://wordribbon.tips.net/T012870_Passing_Custom_Caption_Labels_from_a_Template_to_a_Document_Based_on_the_Template.html</a>). To work around this restriction, use the macro-enabled _Thesis Template.docm_ file.
